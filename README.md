## About Stratum

Stratum is an administrative panel for clients and admins to manage their KVM instances via Proxmox's API.

## Requirements

- PHP 8.0 (required extensions: Fileinfo, JSON, Mbstring, OpenSSL, PDO)
- NGINX or Apache server
